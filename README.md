عرفی پروژه – وب‌سایت رزومه شخصی (اولین پروژه فرانت‌اند)

این پروژه اولین تجربه عملی من در حوزه توسعه فرانت‌اند است که به‌صورت کامل از طریق خودآموزی و بدون شرکت در کلاس، بوت‌کمپ یا دوره‌های حضوری اجرا شده است.

هدف اصلی این پروژه، ساخت یک وب‌سایت رزومه با استفاده از مفاهیم پایه فرانت‌اند و پیاده‌سازی آن‌ها در یک پروژه واقعی و قابل انتشار بوده است، نه رسیدن به یک خروجی کاملاً بی‌نقص از نظر بصری.

ویژگی‌ها و ساختار فنی پروژه

پیاده‌سازی کامل با HTML و CSS خالص بدون استفاده از فریم‌ورک‌ها
با هدف درک عمیق ساختار صفحه، استایل‌دهی و چیدمان عناصر.

استفاده از ساختار معنایی (Semantic HTML)
تفکیک بخش‌ها مانند هدر، سکشن‌های محتوا و فوتر برای خوانایی و توسعه‌پذیری بهتر.

پایه‌گذاری ریسپانسیو بودن صفحات
این پروژه نقطه شروعی برای درک طراحی واکنش‌گرا بوده و در پروژه‌های بعدی به‌صورت بهینه‌تر پیاده‌سازی شده است.

انتشار پروژه روی GitHub Pages
به‌منظور تجربه روند واقعی دیپلوی، مدیریت نسخه و انتشار آنلاین پروژه.

شامل فاصله‌گذاری، سلسله‌مراتب تایپوگرافی و چیدمان اولیه برای ایجاد یک رابط کاربری خوانا
