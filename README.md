این پروژه اولین تجربه عملی من در توسعه فرانت‌اند است که به‌صورت کامل خودآموز و بدون حضور در دوره یا بوت‌کمپ اجرا شده است. هدف آن ساخت یک وب‌سایت رزومه با استفاده از HTML و CSS خالص بوده تا مفاهیم پایه فرانت‌اند و چیدمان عناصر در عمل تمرین شود. پروژه شامل استفاده از ساختار معنایی صفحات، سلسله‌مراتب تایپوگرافی، فاصله‌گذاری مناسب و پایه‌گذاری ریسپانسیو بودن بوده و تجربه انتشار آنلاین از طریق GitHub Pages برای مدیریت نسخه و دیپلوی نیز کسب شده است. این پروژه نقطه شروع من برای ورود به طراحی صفحات واکنش‌گرا و پروژه‌های پیشرفته‌تر بوده است.
